# 📊 E-commerce Sales Dashboard (Excel + Power Pivot)

## 🚀 Overview

This project is an interactive sales dashboard built using Excel, Power Pivot, and DAX to analyze e-commerce data.

## 📌 Features

* KPI Cards: Total Sales, Profit, Orders, Profit Margin
* Sales Trend Analysis (Monthly)
* Category, Region & Segment analysis
* Top 5 Products
* Interactive slicers for filtering

## 🛠 Tools Used

* Microsoft Excel
* Power Pivot
* DAX

## 🧠 Key Insights

* Fashion category generates highest revenue
* East region leads in sales performance
* Top-performing products identified
